## 🛠 Featured Work: Kurdish Keyboard (XKB)
Native Linux kernel support for the Kurmanji layout. 

### ⚙️ Installation Guide
To install the keyboard layout on your Linux system, run the following commands in your terminal:

```bash
git clone [https://github.com/Zinaro/KurdishKurmanjiKeyboard.git](https://github.com/Zinaro/KurdishKurmanjiKeyboard.git)
cd KurdishKurmanjiKeyboard
chmod +x install.sh
./install.sh
```
